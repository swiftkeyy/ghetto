# 🎯 GHETTO VPN - Checklist для запуска

## ✅ Pre-Launch Checklist

### 1. Конфигурация окружения

- [ ] Скопировать `.env.example` в `.env`
- [ ] Заполнить `BOT_TOKEN` от @BotFather
- [ ] Установить сильный `SECRET_KEY` (32+ символов)
- [ ] Установить сильный `JWT_SECRET_KEY`
- [ ] Настроить `POSTGRES_PASSWORD`
- [ ] Добавить свой `ADMIN_USER_IDS`
- [ ] Проверить все переменные окружения

### 2. Telegram Bot

- [ ] Создать бота через @BotFather
- [ ] Настроить команды бота (`/setcommands`)
- [ ] Получить и добавить токен в `.env`
- [ ] Получить свой Telegram ID (через @userinfobot)
- [ ] Добавить ID в `ADMIN_USER_IDS`
- [ ] Протестировать команду `/start`

### 3. База данных

- [ ] Запустить PostgreSQL (`docker-compose up -d postgres`)
- [ ] Проверить подключение
- [ ] Применить миграции (`alembic upgrade head`)
- [ ] Создать первого администратора
- [ ] Проверить таблицы в БД

### 4. VPN Серверы

- [ ] Настроить минимум 1 VPN сервер (WireGuard/VLESS)
- [ ] Добавить сервер в БД через админку
- [ ] Проверить доступность сервера
- [ ] Настроить firewall правила
- [ ] Протестировать подключение

### 5. Платежные провайдеры

**Stripe** (опционально):
- [ ] Создать аккаунт Stripe
- [ ] Получить API ключи (test/live)
- [ ] Добавить в `.env`
- [ ] Настроить webhook
- [ ] Протестировать тестовый платеж

**Crypto** (опционально):
- [ ] Настроить CryptoCloud/CryptoBot
- [ ] Получить API ключи
- [ ] Добавить в `.env`
- [ ] Протестировать платеж

**ЮKassa** (опционально):
- [ ] Создать магазин в ЮKassa
- [ ] Получить Shop ID и Secret Key
- [ ] Добавить в `.env`
- [ ] Протестировать платеж

### 6. Backend

- [ ] Установить зависимости (`pip install -r requirements.txt`)
- [ ] Запустить backend (`docker-compose up -d backend`)
- [ ] Проверить health endpoint (`/health`)
- [ ] Проверить API docs (`/docs`)
- [ ] Протестировать основные endpoints

### 7. Admin Panel

- [ ] Установить зависимости (`npm install`)
- [ ] Настроить `.env.local`
- [ ] Собрать проект (`npm run build`)
- [ ] Запустить (`docker-compose up -d admin`)
- [ ] Открыть в браузере (`http://localhost:3000`)
- [ ] Авторизоваться как администратор

### 8. Мониторинг

- [ ] Запустить Prometheus (`docker-compose up -d prometheus`)
- [ ] Запустить Grafana (`docker-compose up -d grafana`)
- [ ] Открыть Grafana (`http://localhost:3001`)
- [ ] Сменить пароль по умолчанию
- [ ] Импортировать дашборды
- [ ] Проверить метрики
